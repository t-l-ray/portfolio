The following are some of the project and products in which I've had a lead role.

Model 27

Wearable radiation detector for first responders for detecting gamma and neutron radiation.

Low power device with rugged construction. Utilized Bluetooth low energy (BLE) for communications with iOS/Android App for live telemetry.

Primary focus was circuit design, heavy influence on overall mechanical design and UI/UX.

Automatic Testing for Model 3100 Validation

Batch testing of up to 30 instruments at once for passing U.S. Navy MilSpec standards.

Data collected every 2 seconds for tests that could run for weeks at a time.

Data capture interface is built primarily with Node-RED and python and provides live and historical view of each DUT during test so progress can be monitored remotely.

All data is stored in a MariaDB (MySQL) for post processing.

Also built post processing toolkit in JupyterHub using python (pandas, matplotlib) that empowered non-programmer testers to process and present data as needed.

Reduced testing time by at least 4x and identified several flaws that would have been missed with manual testing.

Model 3277

Alpha beta desktop frisker or hand foot monitor for personnel at entry/exit points. Detection circuits are modular and can be configured for 1-3 detectors depending on application.

Powered by SBC running Linux with touch screen interface.

Model 30

Modular radiation instrument intended to upgrade old 2 handed systems to a modern digital single handed operation.

First "professional" product out of college. Sole person on circuit, layout, and firmware.

Calibration is fully digital and instead of analog potentiometers. The protocol I wrote for this has been standardized and reused in newer instruments with a common software package.

Sumo Bot

Educational robot designed for teaching students electrical and software basics.

Includes 16x16 LED matrix, 4x 7 segment number display, 2x DC motor drivers, audio, 5x button control pad, microphone, XBee communications, LiPo battery charger, servos, prototyping area, and edge detection sensors around the bottom.

I designed both the schematic and layout for the various PCBs.

Sumo Bot Controller
